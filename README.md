# Days Spent

https://iamsonukushwaha.github.io/days/


A tiny Progressive Web App (PWA) that shows how many days you've spent in the current year. It displays a progress bar, a month-by-month calendar, highlights days up to today, and can generate a shareable image with a quote and credit.

## Features ✅

- Year progress bar showing days completed and percentage
- Month grids with weekday headers (Monday-first) and proper day alignment
- Click a day to open a Wikipedia page for that date
- Shareable image containing quote, progress, and author credit
- Offline support via a service worker (caches app shell)
